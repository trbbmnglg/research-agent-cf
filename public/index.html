About this app

Personalized AI News Agent — a self-correcting research assistant that runs daily and learns from its own history to cover new ground each run.

How it works — the agentic loop
plan → search → score → synthesize → reflect. After drafting a briefing, the reflect node critiques its own coverage, lists what's missing, and loops back to run targeted searches — repeating until coverage is solid or it hits the max research passes cap.

Cross-run learning (the DB-powered upgrades)
Every run is stored in Cloudflare D1. Before each new run the agent reads its own history to become smarter:

  • Dynamic lookback window — cron runs compute how many days have passed since the last briefing on the same topic and set the search window exactly. No stale re-coverage, no new-content gaps.
  • Gap carryover — the reflect node records blind spots it found in each draft. The next run's plan node reads these and folds them into its initial queries, so the agent improves its search strategy across runs.
