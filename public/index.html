About this app

Personalized AI News Agent — an autonomous research agent that runs daily, emails a curated briefing to your inbox, and learns from its own history to cover new ground each run.

How it works — the agentic loop
researcher → score → synthesize → reflect. The researcher node binds a search_news tool to Claude and lets the LLM decide what to search — it picks queries, calls the tool in parallel, reads results, and calls again if coverage looks thin. After drafting, the reflect node critiques its own coverage, lists what's missing, and loops back for targeted re-searches — repeating until coverage is solid or it hits the max research passes cap. The finished briefing is then delivered to your inbox via Resend.

Cross-run learning (DB-powered)
Every run is stored in Cloudflare D1. Before each new run the agent reads its own history to become smarter:

  • Dynamic lookback window — cron runs compute how many days have passed since the last briefing on the same topic and set the search window exactly. No stale re-coverage, no new-content gaps.
  • Gap carryover — the reflect node records blind spots it found in each draft. The next run folds these into its first researcher pass, so the agent improves its search strategy across runs without any explicit training.
